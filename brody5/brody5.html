List of Programming Languages

  • HTML
    1. Hyper Text Markup Language
    2. Used to take a page of blank text and turn it into a well designed website.
  • Java
    1. Object Oriented Programming Language
    2. Written at Sun Microsystems
    3. Easily Transferrable Between Computers
    4. Used For:
      • Web Applications
      • Web Services
      • Android Applications
  • Python
    1. Very Easy To Learn
      • Easy to Read
    2. Written in C
    3. High Level Language
    4. Used For:
      • Web Development
      • Game Programming
      • Desktop GUIs
      • Scientific Programming
      • Network Programming
  • JavaScript
    1. Used to Enhance the Function of Webpages
    2. Similar to Other Languages Such as Python and Java
    3. Used For:
      • Creating Text Boxes
      • Creating Buttons
      • Creating Popups
      • Changing Text On a Page

    1. Server-Side Scripting Language
    2. Used for Web Development
    3. Database Organization
      • E-Commerce
      • Project Management
      • Graphical User Interface (GUI)
      • Online Communities
      • Facebook Apps
      • Mailing Lists
