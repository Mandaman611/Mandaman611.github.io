Web Programming Projects

Brin Brody - 2017

What is HTML?

An introduction to the HTML language, including use of some of the most basic tags.

Introduction to CSS

An idroduction to the styling language of CSS. Includes both internal and external CSS.

Links

An introduction to the use of link tags, including the different kinds of links.

Movie Summary

Using previously acquired knowledge, a website is created about a favourite movie, including the cast, snynopsis, and other information.

Lists

An introduction to the list tags, including ordered and unordered lists. Nested lists were introduced, as well as custom bullets.

CSS Challenge 1

A CSS challenge in which the user may only change CSS code to acquire the desired result. Basic changes in font and background colour.

CSS Challenge 2a

Making Shakespeare's Sonnet #18 appear beautiful with CSS, in a centered, bulletless list format. Colour changes included.

CSS Challenge 2b

Further changing Shakespeare's 18th Sonnet with colour, alignment, and an introduction to div styling.

Marking Period 1 Final Project

A 4 page fully linked site on a topic of the student's choosing. Includes images, lists, CSS styling and JavaScript.

CSS Challenge 3

Using ONLY CSS to make a two column page using div styling. Further modification of Shakespeare's 18th sonnet.

Tables

An introduction to HTML tables and their associated tags in order to create an informational, and aesthetically pleasing webpage.

Biography

Using skills acquired in previous assignments, students create a biography page about a historical or popular figure.

Forms

An introduction to form objects. A webpage that collects basic information from the user including a clear button.

CSS Challenge 4

A continuation of the third CSS Challenge, with an added header and footer. Uses div styling.

Three Column Styling

Students created a webpage on a topic of their choosing and using CSS, presented information in three columns.

Midterm Project

Using all previously acquired skills, students created a website about their school schedules. Includes more informational subpages.

Slides

Using JavaScript, students choose images about a specific topic and make it into a clickable slideshow.

Dropdown Menu

Using JavaScript, students created a website about a topic of their choosing, linking to different parts of the website using a dropdown menu.

Snake

Modifying a previously made snake game, students changed the game using JavaScript to better fit their needs.

Math Operators

Modifying a previously made mathematics game, students changed the game using JavaScript to increase its complexity and difficulty.

Guessing Game

Using JavaScript arrays, students created a guessing game on a topic of their choosing, including different hints.

Collage

An introduction to the use of the z-index and position properties. Students took pictures on a topic of their choosing and created a collage.

Image Maps

An introduction to image maps, in which a user can select certain places on the same image in order to link to a certain location.

About Me

Students create an assignment all about themselves using all previously acquired skills.

Guessing Game (Event Handlers)

Using event handlers, students create a guessing game for a user about an object of their choosing.
